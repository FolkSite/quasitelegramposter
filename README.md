# quasiTelegramPoster
quasiTelegramPoster is a simple plugin that sends  new resources to Telegram channel.

## Requirements
* MODX Revolution
* Telegram Bot
* Telegram Channel

## Events
* OnDocFormSave
* OnDocPublished
* OnResourceAutoPublish
